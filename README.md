# 📊 AI Equity Research Tool
This project is an AI-powered Equity Research Tool that allows users to analyze stocks and get structured insights. Users can input a stock symbol (e.g., TCS.NS, INFY.NS), and the system fetches real-time financial data and generates analysis including overview, pros, risks, and investment suggestions.


## 🚀 Overview
This project is an AI-powered Equity Research Tool that allows users to analyze stocks and get structured insights. Users can input a stock symbol (e.g., TCS.NS, INFY.NS), and the system fetches real-time financial data and generates analysis including overview, pros, risks, and investment suggestions.

---

## 🧠 Features
- 📊 Real-time stock data using yfinance  
- 🤖 AI-based analysis (simulated)  
- 🌐 Interactive UI using Streamlit  
- 📈 Supports multiple stock symbols  
- ⚡ Fast and simple interface  

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- yfinance

## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/Vaishnavi-C123/Equity-Research-Tool


2. Navigate to the project folder:

cd ai-equity-research-tool


3. Install dependencies:

pip install -r requirements.txt


4. Run the application:

streamlit run app_ui.py
